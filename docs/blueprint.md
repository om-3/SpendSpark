# **App Name**: SpendSpark

## Core Features:

- User Authentication & Profile: Secure user registration and login, allowing users to manage their profiles.
- Expense Logging & Categorization: Users can input their expenses, assign categories (e.g., food, transport, entertainment), and view historical transactions.
- Subscription Management: Users can add, edit, and delete recurring subscriptions, including name, cost, billing cycle (monthly/yearly), and next renewal date.
- Dashboard Overview: A personalized dashboard displaying summary cards for total monthly spend, upcoming subscription renewals, and current active subscriptions.
- Subscription Renewal Alerts: Visual highlights on the dashboard and via email for subscriptions renewing within a custom-definable threshold (e.g., 3 days).
- Interactive Spending Analytics: Generate dynamic charts (e.g., pie chart for category-wise spending, line graph for monthly trends) using a client-side charting library to visualize financial data.
- AI-Powered Financial Insights tool: An AI tool that analyzes spending patterns and subscription data to identify potential 'subscription leaks' (unused but active services) and suggests areas for budget optimization or savings.

## Style Guidelines:

- Primary Color: #7B68EE (a medium slate blue). This color is chosen to convey reliability and modern elegance, aligning with the precision of financial tracking while remaining visually distinct and professional.
- Background Color: #1A1A2E. A very dark, slightly desaturated indigo background maintains the modern, sleek aesthetic of a dark theme, ensuring content stands out without being stark.
- Accent Color: #C287FA (a light magenta-purple). This vibrant, yet harmonious, accent color provides strong contrast for interactive elements, calls-to-action, and highlights, drawing the user's attention where needed within the dark theme.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired feel in titles and prominent text. Body font: 'Inter' (sans-serif) for clean readability and objective presentation of financial data.
- Utilize minimalist, modern outline icons to complement the sleek UI, ensuring clarity and unobtrusiveness for navigation and actions. Icons should adapt to the dark theme.
- Employ a responsive layout with a fixed sidebar for primary navigation and a main content area. Implement rounded corners for cards, input fields, and buttons to enhance the 'modern UI' feel. Maintain generous spacing to prevent visual clutter and improve readability.
- Incorporate subtle hover effects on interactive elements and smooth transitions for data updates (e.g., chart re-renders, summary card changes) to create an engaging and polished user experience without distractions.
